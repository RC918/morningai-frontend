# Morning AI Frontend

Morning AI 官方網站前端專案，基於 Next.js 開發。

## 🚀 技術棧

- **Framework**: Next.js 14.0.4
- **Language**: TypeScript
- **Deployment**: Vercel
- **Node Version**: 18 LTS

## 📦 開發環境設置

### 安裝依賴

```bash
npm install
```

### 開發模式

```bash
npm run dev
```

### 建置專案

```bash
npm run build
```

### 啟動生產環境

```bash
npm run start
```

## 🌐 環境變數

- `NEXT_PUBLIC_STAGE`: 環境標識 (prod/staging)
- `NEXT_PUBLIC_API_BASE`: API 基礎 URL
- `NEXT_PUBLIC_BUILD_ID`: 構建 ID

## 🚀 部署

專案自動部署到 Vercel，支援：
- 生產環境: https://morningai.me
- API 端點: https://api.morningai.me
